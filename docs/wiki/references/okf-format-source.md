# Source

This wiki has been shaped as an OKF v0.1 bundle.

# Applied Conventions

- Concept documents have YAML frontmatter with a `type`.
- Root and directory `index.md` files list available concepts.
- `log.md` records changes to the bundle.
- Relationships are represented with normal markdown links.
